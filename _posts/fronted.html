What I've learned about coding

What I learned about coding is that at its core, it is the act of writing very detailed instructions for a computer to read and perform the actions that are given in code. Programming languages are in effect a way for people to be able to write down these instructions, then a computer translates the code into binary (computer language) in order to perform the actions that were written. I learned that computers are actually not intelligent at all, they can process information very quickly but they cannot create things. That is where programmers come in and basically create blueprints with code to create actual things, like apps, video games, etc. Code is written on an IDE (Integrated Development Environment), which is a software application that allows programmers to write code. In the early days code used to be written on punch cards that a computer could translate to do tasks. Later on programming languages were created to allow programmers to make coding easier and to be able to have a computer do more complex tasks.

Coding not only has helped create software for entertainment purposes, but it also helps make our modern lives more convenient and accessible. Some of the things that exist because of coding are the following:

  • Apps (mobile & desktop)
  • Video games
  • Websites
  • Text-to-speech software
  • Operating systems

HTML

HTML stands for HyperText Markup Language, it is a language used for creating websites. When creating a website HTML is responsible for actually holding the content within the website and establishing the structure of the website. Lets use a blog website as an example, a lot of the content in a blog is written, so it would be held inside a p tag/element. This element represents a paragraph or any line of words. Since those blog posts are the main part of that website those p tags would be held inside the main tag. As far as structure goes, it is very common for most websites to have an area where they can press buttons or links to navigate to other parts of the website. These links or buttons would be held in a nav tag, so users can navigate to different parts of the website.

Now it is not as simple as writing what you want in these tags. With HTML and other technologies there are rules/syntax that is used when coding in HTML. For example, going back to the navigation on a website, you can't simply write in plain text your links or a button inside the nav tag. If you are creating a list of different links you would have to declare that by nesting another group of tags within the nav tag. For links there are two options, you could use anchor tags represented by the a tag or you could create a list first with the ul (unorganized list) and li (list items) tags. If you are making a list then the a anchor tags are put within the li tags. In this example, written content is held within the p tag. The structure of the website is established by having a navigation bar that could take users to different parts on the site, which is held within the nav tag.

CSS

CSS stands for Cascading Style Sheets, it is a language used for styling and editing the appearance of websites. It is called Cascading Style Sheets due to the way the files in CSS are processed. With CSS you can take a very basic website and really make it pop with color and design choices. CSS is used with HTML, but it has different syntax than HTML. In CSS for the most part you are listing properties that you want to change and style. In the beginning it can seem daunting trying to remember all the different properties, but just like with HTML once you start coding on a regular basis it becomes easier to remember properties and proper syntax. Usually CSS is is the second step when coding a website because you need content and structure first using HTML, to then use CSS to style the website.

CSS uses selectors such as body or main to select which element is going to be styled and then properties like font-size are chosen and given values. Using the same example of a simple blog website, lets look into how CSS can be used to style a website. Lets say that a coder wants to style the words in p tags, they would declare it in CSS like so: p {} and then within the curly braces the properties and their values can be written. An example of a property and its value would look like this: font-size: 20px;. This would give words in p tags a font size of 20 pixels. When it comes to styling the navigation bar you could declare the anchor tag like so: a {}. Then declare a property and a value such as: color: azure;, this would change the color of the links in the navigation bar to an azure color. CSS could also be used for creating animations using the @keyframes property, but the animations are still for styling a website. You would not use it to create a cartoon show for example. Most importantly CSS could also make websites more accessible by using the @media tag, which can help make a website function on any device.
